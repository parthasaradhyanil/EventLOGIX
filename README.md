
EventLOGIX
The Official Campus Event & Hackathon Portal

EventLOGIX is a lightweight, responsive Single Page Application (SPA) designed to help students discover, track, and register for campus events, workshops, and hackathons. It features a modern dual-view system, allowing users to browse events as detailed cards or visualize them on an interactive monthly calendar.

 Key Features
Dual View System: Seamlessly toggle between a detailed "List View" and a structural "Calendar View."

Smart Memory: Uses browser localStorage to remember the user's preferred view (List or Calendar) across sessions.

Interactive Calendar: A fully functional, dynamic calendar that automatically calculates month lengths, leap years, and starting days.

Fully Responsive: Built with CSS Grid and Flexbox to ensure the portal looks perfect on desktop monitors, tablets, and mobile phones. On mobile, the calendar intelligently transforms into a scrolling list.

Theming Engine: Powered by CSS Variables (:root), allowing developers to change the entire color scheme of the site by editing just one or two lines of code.
